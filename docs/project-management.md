# Gestión del Proyecto

## Herramienta utilizada
Para organizar el desarrollo de "Top 5 de Todo" he usado un tablero Kanban en Trello

## Organización del trabajo
El tablero está dividido en cinco columnas que representan el estado de cada tarea:

- Backlog: tareas pendientes que aún no tienen prioridad inmediata.
- Todo: tareas listas para empezar en el siguiente paso.
- In Progress: tareas en las que se está trabajando activamente.
- Review: tareas terminadas que se necesitan revisar o probar antes de darlas por válidas.
- Done: tareas completamente terminadas y funcionales.

## Organización de las tarjetas
Cada tarjeta representa una funcionalidad o bloque técnico del proyecto. Dentro de cada una hay un checklist con las subtareas técnicas concretas, lo que permite ver el progreso de forma más detallada.
