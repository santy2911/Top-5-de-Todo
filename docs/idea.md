# Idea del Proyecto: Top 5 de Todo

Top 5 de Todo es una aplicación web que permite crear, gestionar y visualizar listas personales de exactamente 5 elementos sobre cualquier tema (películas, series, videojuegos, música, comida, etc.).

## ¿Qué problema resuelve?

A menudo queremos hacer listas de nuestras cosas favoritas, pero hay tantas opciones que es difícil decidirse y organizarlas bien. Esta aplicación soluciona eso al limitar las listas a solo 5 posiciones, obligando a elegir lo más importante, y ofrece una forma rápida y ordenada de crearlos.

## Usuario objetivo

Cualquier persona que quiera organizar y compartir sus preferencias personales: películas favoritas, mejores canciones, top de videojuegos, comidas, series, etc.

## Funcionalidades principales

- Creación de listas: título, categoría y exactamente 5 posiciones.
- Visualización de todas las listas en una lista clara y organizada.
- Eliminación de listas.
- Límite de 10 listas (simulando versión gratuita).
- Persistencia de datos con localStorage (se guardan aunque se cierre el navegador).

## Funcionalidades opcionales

- Edición de listas existentes (cambiar título, categoría o posiciones).
- Búsqueda de listas por título.
- Generar un texto listo para copiar y compartir el Top 5.

## Mejoras futuras

- Sistema de usuarios con registro e inicio de sesión.
- Perfil público para mostrar las listas a otros usuarios.
- Plan Premium para eliminar el límite de listas y añadir más funcionalidades.
